How to Use this Calculator



General Input Guidelines

This calculator exclusively uses integers and fractions for the sake of accuracy. Entering decimal values will result in an error.
For fractions, entering the numerator, then a forward slash, then the denominator will suffice.
Parentheses can only be used as specified for the vector calculator.
Entering values with non-numerical symbols (with the exception of the proper use of the forward slash character) will result in an error.
